## Live Demo

See it live [here](https://shashiirk.github.io/todos).

## Get Started

Clone this repository to your local machine

```
$ git clone https://github.com/shashiirk/todos.git
```

Install all the dependencies

```
$ npm install
```

Start the application

```
$ npm start
```
